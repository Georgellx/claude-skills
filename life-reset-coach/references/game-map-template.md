# "人生游戏地图"产出模板

收尾时，把对话挖出的东西填进这个模板，生成一份用户能每天照打、每周更新的 Markdown 文档。
**用用户自己说过的话填**，别套通用励志词。日期用对话当天。

---

```markdown
# 人生游戏地图 · {用户名/可省略}

> 日期：{YYYY-MM-DD}
> 用 Dan Koe《一天重启人生》的框架，通过一场引导对话整理。每天开工前看一眼，每周回来更新。

## 一句话压缩

- 🩸 我绝不允许我的人生变成：**{反愿景一句话}**
- 🏆 我正在建造的方向：**{愿景一句话，明知会进化}**
- 🧬 我是那种 **{身份陈述}** 的人。

## 🎮 游戏地图

| 部件 | 我的内容 | 游戏机制 |
|---|---|---|
| 🩸 反愿景 | {绝不想再经历的人生，越具体越好} | 赌注 |
| 🏆 愿景 | {想要的、会边做边长的理想生活} | 怎么赢 |
| 🎯 一年主线 | {一年后为真的一件具体、可判输赢的事 + 数字} | 主线任务 |
| ⚔️ 一个月 Boss 战 | {这个月要学的技能/要做出的东西} | Boss 战 |
| 🗡️ 每日杠杆 | {每天推进的 2-3 个关键动作 + 那条死规矩} | 日常支线 |
| 📏 规则 | {不愿牺牲的约束：如养家、健康/精力上限} | 规则 |

## 一年主线的拆解（想法是有层级的，不能一步登天）

{把主线数字拆成更小的台阶，并标出"真正的第一小步"。}

## 每日唯一判定标准

> 今天结束时，世界上有没有多出一个"别人能看到、能反应"的东西？
> 有 = 我在掌舵；没有 = 我今天还停在"想"，没动。

⚠️ 最大的敌人不是没能力，是 **{用户的具体"安全区"，如：找工作/思考}** 会悄悄吃光精力。
守住"先动、产出一个能被检验的东西"这条线。

## 今天就做的第一个动作

{一个今天能完成、且会产出"能被反应的东西"的最小动作，适配用户性格。}
```

---

## 填写提醒

- **反愿景**：如果用户已身处困境，直接用他的真实处境，别让他"想象"。这是最强的燃料。
- **一年主线**：必须逼出数字或具体事件，否则不是目标，是愿望。
- **规则**：把用户的现实约束（养家、疾病、精力）写成"规则"而非"失败"——这是尊重，也符合作者原意（约束=激发创造的规则）。
- **每日动作**：适配性格。内向者→异步输出（写作/做原型/发布）而非实时社交。
- 收尾别忘了**提议存到本地或记忆**，方便续接。
