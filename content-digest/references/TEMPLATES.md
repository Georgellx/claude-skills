# 模板集（写文件前先读这里，frontmatter 照此，别写坏写法）

通用 frontmatter 规矩：`type` 只用 8 值词表（官方快照/文章笔记/书籍/自我认知/产品想法/复盘/索引/提示词）；`tags` 块列表；`related_notes` 加引号块列表 + 真实文件名；空字段删整行，不留空占位。

---

## 1. 文章笔记模板 → `知识资产\文章笔记\<标题>.md`

```markdown
---
title: 中文标题
original_title: 英文/原标题        # 没有就删掉这一行，别留空串
author: 作者                      # 缺就删行
source: 来源名（如 生财有术 / 微信读书 / 某博主）
source_url: "https://..."         # URL 一律加引号；没有就删行
date: YYYY-MM-DD
type: 文章笔记
tags:
  - 主题概念标签
  - english-tag
related_notes:
  - "[[已有真实文件名]]"            # 0-3 条；一条都没有就把 related_notes 整段删掉
---

# 文章摘要
<紧凑的中文摘要>

# 核心观点
- <最强、最可复用的观点，不是每个细节>

# 可复用句子/表达
- <值得复用的句子/框架/措辞>

# 方法解读与验证   ← 仅当文章有方法论/可验证论断时加；与内容体量成比例
> 价值判断：<A/B/C 档 + 一句理由>
<大白话重述 + 真伪/适用性点评：哪些有理论支撑、哪些是个人经验、哪些是营销话术。B/C 档在这里放干净重写版或提炼>

# 关联笔记
- [[已有真实文件名]]：<为什么连>

# 原文
<A/B 档：完整保留原文（含必要图片 markdown）；提示词部分用指针双链代替全文复制。
 C 档：来源链接 + 关键引用即可，不灌全文。>
```

---

## 2. 产品想法模板 → `知识资产\产品想法\<短名>.md`

```markdown
---
title: <想法全名>
author: george
source: <来源：博主名 / 某文章标题 / 个人灵感>
idea_source: 复刻                  # 复刻（抄博主已验证打法）/ 原创（自己冒的）
date: <YYYY-MM-DD>
type: 产品想法
status: 待孵化                     # 待孵化 → 做原型 → 已上线 → 归档
tags:
  - 产品想法
  - <按内容补 2-4 个，如 AI视频/声音克隆/自动化>
related_notes:
  - "[[00-想法总览]]"
  - "[[相关真实文件名]]"            # 没有就删这一行
---

# 💡 <想法全名>

**一句话**：<用一句话说清这是什么>

## 已验证样本 / 复刻证据   ← 核心。复刻类必填，原创类填"需求验证"
- **谁的打法 / 来源**：<博主、文章、平台>
- **成了没有（真实数据）**：<粉丝数 / 收入 / 销量 / 具体案例。文章里有就抽出来；没有就写"待验证"，不编>
- **这是"想法"还是"真实需求"**：<有人在抱怨/在找/在付钱吗？需求 > 想法。没把握就写"待验证：去 X 平台找样本">

## 解决什么 / 为什么有意思
- <痛点、人群、为什么值得做；和我的方向/热情什么关系>

## 怎么做（技术路径设想）
- <实现思路、可能用到的工具、产品形态。能写几步写几步>

## 商业化想象
- <怎么赚钱 / 怎么积累，没想好写"待补">

## 风险与难点
- <最大的坑：版权？平台风控？质量？如实写>

## 下一步（如果要验证）
- <最小验证动作。固定套路：先手动跑通一条完整流程，做出 1 个能发的成品，再谈产品化>
```

**status 取值**：`待孵化`→`做原型`→`已上线`→`归档`。默认 `待孵化`；用户说"已做了原型/在做了"就写对应状态，正文顶部用引用块注明并双链到相关复盘。
**status emoji（写进索引看板）**：💡速记 / 🌱待孵化 / 🛠️做原型 / 🚀已上线 / 🗄️归档。

## 3. 提示词收录格式（追加到对应 `0X-*.md` 末尾，追加不覆盖）

````markdown
## <提示词名>

用途：<一句话说清用在什么场景、怎么用>。来源 [[<文章笔记标题>]]。

`````text
<提示词正文原样保留>
`````
````

> 外层用**四个反引号**围栏（库里既有约定，因为提示词内部可能含三反引号）。上面示意里的五反引号在实际写入时改成四反引号。
